# Smart Infrastructure Monitoring System

Smart Infrastructure Monitoring System is an AI-powered predictive maintenance platform developed using Spring Boot, Thymeleaf, Java, and MySQL. 

The system monitors infrastructure assets by analyzing temperature, vibration, and pressure data to calculate health scores and predict failure risks. 

It classifies assets into Healthy, Warning, or Critical states using anomaly detection logic and displays results on an interactive dashboard with live simulation and trend visualization.

## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- MySQL
- HTML, CSS, JavaScript
- Chart.js

## Features
- Real-time simulation
- Health score calculation
- Failure probability prediction
- Anomaly detection
- Interactive dashboard
- Trend visualization
